MCA FSWDT Lab

Subject Details :-

  • Subject Code : CA2009
  • Subject Name: Internet & Web Technology
  • Credit : 4
  • Grading Pattern: 5
  • Course Outcome (CO):

I. Demonstrate the capability to classify its elements and attributes.

II. Create web pages using XHTML and Cascading Style Sheets.

III. Build dynamic web pages using JavaScript (Client Side programming).

IV. Create XML documents and Schemas.

Course Description :-

Module-I: Introduction to Internet- Basics concepts of Internet, A Brief Overview of TCP/IP and its Services, Communication on the Internet, Internet Diversity, Establishing Connectivity on the Internet, Client/ IP Address, Web Browsers, Web Servers, Web Clients, Domain Registration.

Module-II: Introduction to HTML. HTML Tags and Attributes, Commonly Used HTML Commands, Title and Footers, Text Formatting, Text Style and Font Assignments, Text Effects, Document & Window Parts, Lists, HTML and Audio Tags, HTML Image Tag, HTML Hyperlink Tag, HTML Marquee, HTML Frames (Horizontal & Vertical Frames), HTML Table Tag, HTML Scrolling Attribute in Image Tag, Hypertext, Hyperlink and Hyperlinks. Links, Anchors and URLs Lists, Adding Graphics to HTML Documents, Creating Tables, Forms, Linking Documents, Creating Links, Frames and Frame sets.

Module-III: Introduction to Java Script - Java Script in Web Pages, Advantages of Java Script, Advantages of Java Script, Data Types and Variables, Type Casting - Java Script Functions, Objects, Arrays, Data Types and Function, Function, Creating Objects, even and Array, even handling, Introduction to AJAX, VB Script.

Module-IV: Introduction to XML. HTML, XML, and HTML, Modeling XML Data, Storing XML with XSL, XHTML. Introduction of CGI -Introduction of COM - DCOM, Introduction of ASP, Introduction of PHP and MySQL.
